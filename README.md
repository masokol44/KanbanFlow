📋 KanbanFlow — менеджер задач (канбан-доска)
Профессиональная система управления задачами с поддержкой канбан-досок, приоритетов, дедлайнов, тегов и статистики.
Реализована на 7 языках программирования для демонстрации различных подходов к организации рабочего процесса.

🧠 Концепция
KanbanFlow — это не просто список задач. Это полноценная канбан-доска для управления проектами и личными делами:

✅ Колонки (столбцы): «К выполнению», «В работе», «На проверке», «Готово» (можно настраивать).

✅ Задачи с заголовком, описанием, приоритетом (высокий/средний/низкий), дедлайном, тегами.

✅ Drag & drop в GUI-версиях для перемещения задач между колонками.

✅ Фильтрация по статусу, приоритету, тегам, поиск по тексту.

✅ Статистика — количество задач в каждой колонке, сгоревшие дедлайны, среднее время выполнения.

✅ Экспорт/импорт в JSON для совместной работы.

✅ История изменений — логирование действий с задачами.

✅ Уведомления о приближающихся дедлайнах (при запуске).

✅ Цветовая индикация приоритетов (красный — высокий, жёлтый — средний, зелёный — низкий).

🚀 Как запустить
Каждая версия самодостаточна и хранит данные в локальном файле kanban.json.
Для консольных версий доступен интерактивный текстовый интерфейс с командами.

bash
# Python (Tkinter GUI)
pip install tk
python kanban_python.py

# C++ (Qt)
qmake && make
./kanban_cpp

# Java (Swing)
javac -cp . kanban_java.java && java kanban_java

# C# (WPF)
dotnet run

# Go (консоль)
go run kanban_go.go

# Rust (консоль)
cargo build --release && ./target/release/kanban_rs

# Node.js (консоль)
node kanban_js.js

📦 Содержимое репозитория
Файл	Язык	Особенности
kanban_python.py	Python	Tkinter GUI, drag-n-drop, цветные метки, автосохранение
kanban_cpp.cpp	C++	Qt Widgets, таблицы, фильтры, экспорт в CSV
kanban_java.java	Java	Swing, JTable, диалоги, импорт/экспорт JSON
kanban_cs.cs	C#	WPF, DataGrid, MVVM, цветовая индикация
kanban_go.go	Go	консоль с цветами, удобные команды, фильтрация
kanban_rs.rs	Rust	консоль с termion, цветная печать, сериализация
kanban_js.js	JavaScript	Node.js + readline, интерактивный режим
🔮 Расширенные функции
Кастомизация колонок — добавление/удаление/переименование столбцов.

Подзадачи — разбивка задач на подзадачи с собственным статусом.

Временные метки — время создания, последнего изменения, завершения.

Автоматическое архивирование — перемещение старых выполненных задач в архив.

График сгорания задач (burndown chart) — в планах.
